Web Development Bootcamp

The best course I've taken so far! It teaches it all, from the basics to harder topics, such as NoSQL Databases, React and Web3. There are a couple of introduction modules to blockchain technology and also some exercises around encryption. There are exercises in ALL the modules to promote a deeper understandment of the topics. Some of which I've posted in this portfolio. I am sure this course was a great introduction and it is just the beginning!

Free Code Camp - Responsive Web Design

This course is a good option to become familiar with Web Development concepts. I like to compare FCC to Duolingo. It is a repetitive exercise that aims to create a sort of muscular memory. However, I believe that complementing this knowledge through small exercises or even other courses is necessary in order to really learn it. Despite it's characteristics, it is a good plataform to learn new technologies with a very intuitive interface! And it is free!!
